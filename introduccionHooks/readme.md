tarea 3

Ya hemos dicho que el estado puede almacenar no solamente variables booleanas, sino de cualquier tipo: números, strings e incluso arrays y objetos de complejidad variada.

Puede ser una buena idea escribir todas tus buenas acciones. En este ejemplo, agrega el hook necesario y también podrás empezar a hacer esto.

Este es un momento importante. Cuando actualices un array utilizando una función setter, no utilices push() en el array antiguo. En su lugar, crea un nuevo array, clonando el anterior y utilizando el operador spread (...) o el método slice(); y después, agrega los nuevos elementos al nuevo array.
