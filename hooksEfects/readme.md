ejercicio 1

Los efectos se usan normalmente para que un componente se pueda extender más allá de su ámbito. Por ejemplo, esto puede hacerse si queremos cambiar algo en toda la página. Lo que es más importante, esto solo debería suceder después de que el componente se haya renderizado y no debería interferir de ningún modo con el proceso de renderizado.

En este ejercicio, el componente debería activar un precioso fondo de página.

Agrega el efecto que falta, el cual debería llamar a la función con el argumento correspondiente.
